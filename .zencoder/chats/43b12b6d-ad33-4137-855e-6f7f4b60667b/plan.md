## Workflow for each unit:
1. Generate 50 exercises (25 sentence-building, 25 multiple-choice).
2. Ensure B2 level complexity and neutral tone.
3. Use the `[[English|Spanish]]` format for all strings.
4. Add audio paths following the pattern `audio/b2/unit-X/eY.mp3`.
5. Present the unit to the user for review.
6. Once confirmed, move to the next unit.
