# Product Requirements Document - SEO Title Optimization

## Overview
Optimize SEO titles for a provided list of 400 URLs to improve search engine visibility and user experience.

## Objectives
- **Reduce Title Length**: All titles must be between 60-70 characters.
- **Maintain Context**: Titles must concisely describe the page content.
- **SEO Authority**: Act as an expert to ensure high domain authority through keyword-rich and professional titles.
- **Format Consistency**: Words in titles separated by spaces; URL slugs must use hyphens (for future reference).

## Acceptance Criteria
1. Each URL from the input list must have a corresponding optimized title.
2. New titles must be ≥ 60 and ≤ 70 characters long.
3. Titles must be unique and descriptive.
4. Redundant branding (e.g., repeating "| Focus English") should be minimized to save space while maintaining identity.
