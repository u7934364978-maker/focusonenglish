# Technical Specification - Optimización SEO de Artículo

## 1. Ajustes de Estructura SEO
Se modificará el contenido de `src/content/blog/viajes/guia-definitiva-viajes.md` con los siguientes enfoques:

- **Título (H1)**: Mantener el actual pero asegurar que el slug y la etiqueta de título sean coherentes.
- **Primeros Párrafos**: Reescribir la introducción para incluir "inglés para viajar" de forma orgánica.
- **Encabezados (H2/H3)**: Ajustar los nombres de las secciones para incluir variaciones de palabras clave (ej. "Vocabulario de inglés para viajar en avión").
- **Listas y Tablas**: Mantener el formato bilingüe pero optimizar los textos introductorios de cada fase.

## 2. Inserción de Keywords
- **Introducción**: Inserción de "aprender inglés para viajar".

- **Fase 2-3**: "frases de inglés para hoteles y restaurantes".
- **Fase 6**: "vocabulario de compras para turistas".
