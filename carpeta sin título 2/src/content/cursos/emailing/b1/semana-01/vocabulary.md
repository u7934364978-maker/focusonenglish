# Vocabulary (B1) · Semana 01

> En esta semana priorizamos vocabulario útil para: **adjuntos + aprobaciones + follow-up**.

## 1) Palabras clave (EN → ES)
| EN | ES |
|---|---|
| approval | aprobación |
| draft | borrador |
| feedback | feedback / comentarios |
| comments | comentarios |
| timeline | cronograma |
| attachment | adjunto |
| deadline | fecha límite |

---

## 2) Colocaciones (muy típicas en emails)
| Colocación (EN) | Uso (ES) |
|---|---|
| take a look | echar un vistazo |
| share comments / feedback | compartir comentarios |
| latest version | última versión |
| thanks in advance | gracias de antemano |
| if possible | si es posible (suaviza) |
| just checking | follow-up suave |

> “take a look” + “share comments” = combinación muy natural y profesional.

---

## 3) Mini-frases listas (copy/paste)
- Could you please review the draft?
- Could you please approve it by Friday?
- If possible, by Thursday would be great.
- Just checking if you had a chance to review this.
- Let me know if you have any questions.
- Thanks in advance for your help.
