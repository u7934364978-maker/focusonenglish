# English C1 Course — Advanced Proficiency

> **CEFR Level:** C1 Advanced  
> **Prerequisite:** B2 (Upper-Intermediate) consolidated  
> **Total Units:** 60 units · 10 modules · 6 lessons per unit  
> **Estimated Duration:** 300–360 hours  
> **Exam Compatibility:** Cambridge C1 Advanced (CAE) · IELTS 7.0+ · TOEFL iBT 100+

---

## What You Will Achieve

By the end of this course, you will be able to:

- Understand long, complex texts and recognise implicit meanings, attitude and tone
- Express yourself fluently and spontaneously with little effort in finding expressions
- Use language flexibly and effectively for social, academic and professional purposes
- Produce clear, well-structured, detailed text on complex subjects
- Maintain consistent control of complex grammatical structures with minimal errors

---

## Course Structure

The course is organised into **10 thematic modules** of **6 units** each. Every unit contains **6 lessons** following a consistent pedagogical sequence.

### Standard Lesson Format (per unit)

| Lesson | Skill | Focus |
|---|---|---|
| **L1** | Reading | Complex text comprehension + vocabulary in context |
| **L2** | Listening | Audio input (podcast / lecture / debate) + comprehension tasks |
| **L3** | Grammar | Advanced grammar structure with guided practice |
| **L4** | Vocabulary | Thematic lexis: collocations, phrasal verbs, idioms, AWL |
| **L5** | Writing | Guided writing task with model + independent production |
| **L6** | Speaking + Review | Oral activity (debate / presentation / role-play) + unit review |

## CEFR C1 Alignment

This course covers all five CEFR C1 skill descriptors:

| Skill | C1 Can-Do Descriptor |
|---|---|
| **Reading** | Can understand long, complex texts, whether or not they relate to his/her own field, provided he/she can reread difficult sections |
| **Listening** | Can understand enough to follow extended speech on abstract and complex topics beyond his/her own field |
| **Writing** | Can write clear, well-structured expositions of complex subjects, underlining the relevant salient issues |
| **Speaking** | Can express him/herself fluently and spontaneously, almost effortlessly. Has a good command of a broad lexical repertoire |
| **Grammar** | Maintains consistent grammatical control of complex language, even while attention is otherwise engaged |

## How to Use This Course

**For instructors:**
1. Start with `syllabus.md` to understand the full scope and sequence
2. Read each unit file before the module begins
3. Use `reference/grammar-scope-sequence.md` to track grammar progression
4. Apply rubrics from `assessment/evaluation-criteria.md` for all skill evaluations

**For self-study learners:**
1. Follow units in order — each module builds on the previous
2. Complete all 6 lessons per unit before moving on
3. Use the can-do checklists at the end of each unit to self-assess
4. Unit 60 contains full exam practice for Cambridge CAE / IELTS / TOEFL

**Pacing guide:**
- Intensive (5h/week): ~60–72 weeks
- Standard (3h/week): ~100–120 weeks
- Each lesson is designed for approximately 50–60 minutes of class time
