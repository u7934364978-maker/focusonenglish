### L3 — Grammar: *It* Structures

**Forms:**

| Structure | Example |
|---|---|
| *It is + adj + that + clause* | *It is essential that all staff are informed.* |
| *It is + adj + to-infinitive* | *It is increasingly difficult to maintain work-life balance.* |
| *It is + past participle + that* | *It is widely acknowledged that flexible working increases productivity.* |
| *It + seems/appears/tends + that* | *It appears that remote working is here to stay.* |
| *It is + noun phrase + that* | *It is the responsibility of employers to support mental health.* |
| *It cannot be denied that* | *It cannot be denied that the pandemic accelerated change.* |
| *It would be premature to* | *It would be premature to claim that offices are obsolete.* |

**Use notes:** *It* structures allow impersonal, distanced statements typical of formal register. They avoid personal attribution and are common in academic and professional writing.

### L3 — Grammar: Cause and Result Linkers

**Forms:**

| Function | Formal linkers | Example |
|---|---|---|
| **Cause** | *owing to, due to, as a result of, on account of, in view of* | *Owing to supply chain disruption, prices rose sharply.* |
| **Result** | *consequently, therefore, thus, hence, as a result, thereby* | *Trade volumes fell; consequently, GDP contracted.* |
| **Purpose** | *with a view to, so as to, in order to* | *With a view to reducing inflation, interest rates were raised.* |
| **Logical conclusion** | *it follows that, this suggests that, it can be inferred that* | *It follows that sustained investment is necessary.* |

**Contrast with informal equivalents:**
- Informal: *because of* → Formal: *owing to / as a result of*
- Informal: *so* → Formal: *consequently / therefore / thus*
- Informal: *to* → Formal: *in order to / so as to*

**Exercises:**
1. Replace informal cause/result words with formal alternatives in 8 sentences
2. Link 6 pairs of sentences using an appropriate cause/result linker
3. Write a 100-word economic analysis paragraph using 5 different cause/result linkers

### L3 — Grammar: Business Conditionals

**Forms:**

| Structure | Meaning | Example |
|---|---|---|
| *Provided (that)* | On the condition that | *Provided that funding is secured, the project will launch in Q3.* |
| *On condition that* | Only if | *We will proceed on condition that all parties sign the agreement.* |
| *As long as / so long as* | While the condition remains | *The strategy will work as long as market conditions are stable.* |

| *In the event that* | If (formal) | *In the event that the merger fails, contingency plans are in place.* |

### L3 — Grammar: Gerund vs Infinitive — Advanced Usage

**Verbs with no meaning change (same meaning either way):**  
*begin, start, continue, intend, prefer, attempt, love, like, hate*

**Verbs with meaning change:**

| Verb | + Gerund | + Infinitive |
|---|---|---|
| *stop* | stopped doing (ceased) — *She stopped working.* | stopped to do (paused) — *She stopped to think.* |
| *remember* | remembered doing (recall past) — *I remember locking the door.* | remembered to do (recall to act) — *I remembered to lock the door.* |
| *forget* | forgot doing (past action) — *I'll never forget meeting her.* | forgot to do (failed to) — *I forgot to send the report.* |
| *try* | tried doing (experiment) — *He tried meditating.* | tried to do (made effort) — *He tried to focus.* |
| *regret* | regret doing (sorry for past) — *I regret saying that.* | regret to do (sorry to inform) — *I regret to inform you…* |
| *go on* | went on doing (continued) — *She went on working.* | went on to do (then did) — *She went on to become CEO.* |

**Exercises:**
1. Choose gerund or infinitive in 12 sentences, some with meaning change
2. Write sentences showing the meaning difference for 4 verbs
3. Error correction × 6 — gerund/infinitive errors in professional emails

### L3 — Grammar: Advanced Modal Review

**Target structures:**

| Modal expression | Meaning | Example |
|---|---|---|
| *needn't have + pp* | Did something unnecessary | *You needn't have worried — the presentation was excellent.* |
| *didn't need to + inf* | Wasn't necessary (and may not have done it) | *I didn't need to prepare so much; the questions were easy.* |
| *would rather + bare inf* | Present/future preference | *I'd rather work from home than commute every day.* |
| *would rather + past simple* | Preference for someone else | *I'd rather you didn't work overtime this week.* |
| *had better + bare inf* | Strong advice (or else) | *You had better update your CV before the deadline.* |
| *be supposed to + inf* | Expected/required by rule or agreement | *Employees are supposed to give one month's notice.* |
| *was/were to + inf* | Formal planned future or hypothetical | *The merger was to be announced on Friday.* |

**Exercises:**
1. Complete 8 sentences with the correct advanced modal form
2. Distinguish *needn't have* vs *didn't need to* in 5 sentence pairs
3. Write 5 sentences using different advanced modals in a workplace context
