### L3 — Grammar: Inversion for Emphasis

**Target structures:** Negative adverbials + inversion · *Only when / Not until / Hardly / No sooner*

**Form:**

| Adverbial | Inverted structure | Example |
|---|---|---|
| *Never* | *Never + auxiliary + subject* | *Never have I seen such a powerful speech.* |
| *Rarely / Seldom* | *Rarely does + subject* | *Rarely does a politician speak with such clarity.* |
| *Not only* | *Not only did + subject…* | *Not only did she convince the audience, but she inspired them.* |
| *Under no circumstances* | *Under no circumstances should + subject* | *Under no circumstances should personal data be disclosed.* |
| *Only when* | *Only when + clause + auxiliary + subject* | *Only when the results were published did we understand the scale.* |
| *Hardly / Scarcely* | *Hardly had + subject + pp… when* | *Hardly had she started speaking when the applause began.* |
| *No sooner* | *No sooner had + subject + pp… than* | *No sooner had the debate ended than controversy erupted.* |

**Use notes:**
- Inversion is used in **formal and literary language** to add emphasis
- Common in academic writing, speeches, and journalistic prose
- The auxiliary verb is placed **before** the subject (like a question form)

1. **Opening hook** — dramatic statement, question, or quotation
2. **Thesis** — your clear position
3. **Argument 1** — with example or evidence
4. **Argument 2** — with example or evidence

### L3 — Grammar: Complex Noun Phrases

**Target structures:** Pre-modification chains · Post-modification · Nominalisation

**Forms:**

| Type | Structure | Example |
|---|---|---|
| Pre-modification | adjective(s) + noun | *a long-standing, culturally embedded communication norm* |
| Post-modification (relative clause) | noun + relative clause | *the signals that convey distrust* |
| Post-modification (prepositional phrase) | noun + prep phrase | *the study of proxemic behaviour in professional settings* |
| Post-modification (participle phrase) | noun + -ing/-ed phrase | *researchers studying the impact of gaze… / findings published last year…* |
| Nominalisation | verb/adj → noun | *to communicate → communication / to analyse → analysis* |

**Use notes:**
- Complex noun phrases allow **density of information** typical of academic writing
- Nouns can be pre-modified by multiple adjectives, noun adjuncts, and participial forms
- Nominalisation is key to academic register: *The scientists discovered → The discovery by scientists*

**Exercises:**

### L3 — Grammar: Advanced Discourse Markers

**Target structures:** Markers for organisation, concession, contrast, addition and exemplification in formal register

**Categories:**

| Function | Markers | Example |
|---|---|---|
| **Introducing** | *With regard to… / As far as X is concerned… / Turning to…* | *With regard to communication styles, high-context cultures differ markedly…* |
| **Adding** | *Furthermore / Moreover / What is more / In addition / Besides* | *Moreover, the cultural gap was exacerbated by poor preparation.* |
| **Contrasting** | *Notwithstanding… / In spite of this / That said… / Nevertheless* | *Notwithstanding these challenges, the merger proceeded.* |
| **Conceding** | *Admittedly / Granted / It must be acknowledged that* | *Admittedly, the guidelines were unclear.* |
| **Exemplifying** | *To illustrate… / A case in point is… / Take, for example…* | *A case in point is the breakdown in negotiations described above.* |
| **Concluding** | *In light of the above… / All things considered… / To sum up…* | *In light of the above, it is clear that…* |
| **Qualifying** | *To some extent / In so far as / With certain reservations* | *To some extent, the problem was foreseeable.* |

**Exercises:**

### L3 — Grammar: The Subjunctive in Formal English

**Target structures:** Present subjunctive · *Were* subjunctive · Fixed subjunctive expressions

**Form:**

| Type | Structure | Example |
|---|---|---|
| Present subjunctive | *It is + adjective + that + subject + base form (no -s)* | *It is essential that every voice be heard.* |
| Mandative subjunctive | *verb of recommendation + that + subject + base form* | *I suggest that the committee review its procedures.* |
| *Were* subjunctive | *If + subject + were (not was)* | *If this were to happen, the consequences would be severe.* |
| Fixed expressions | *Be that as it may / Far be it from me / Suffice it to say* | *Be that as it may, the decision has been made.* |

**Use notes:**
- The subjunctive is increasingly rare in informal English but remains in formal written and spoken registers
- Common after: *suggest, recommend, propose, demand, insist, require, urge, vital, essential, important, necessary*
- *Were* subjunctive is preferred over *was* in formal and conditional contexts

**Exercises:**
1. Complete 8 sentences with the correct subjunctive form
2. Identify 5 sentences where subjunctive vs indicative changes the register
3. Write a formal proposal paragraph (80 words) using at least 3 subjunctive structures

### L3 — Grammar: Ellipsis and Substitution

**Target structures:** Ellipsis (omitting repeated elements) · Substitution (*do so, one, ones, such*)

**Ellipsis forms:**

| Type | Example | Without ellipsis |
|---|---|---|
| Verb phrase ellipsis | *She can speak Mandarin, and so can he.* | *…and he can speak Mandarin too.* |
| Noun phrase ellipsis | *I'd like the large coffee, not the small.* | *…not the small coffee.* |
| Clausal ellipsis | *"Are you coming?" "I think so."* | *"I think I am coming."* |

**Substitution forms:**

| Substitute word | Replaces | Example |
|---|---|---|
| *do so* | verb phrase | *She wanted to leave early and she did so.* |
| *one / ones* | singular/plural noun | *I prefer the digital version to the printed one.* |
| *such* | noun phrase with classifier | *Language change is inevitable. Such changes cannot be stopped.* |

**Exercises:**
1. Identify all ellipsis and substitution in a short text × 8 instances
2. Rewrite 6 sentences using ellipsis or substitution to avoid repetition
