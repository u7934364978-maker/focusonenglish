# Module 1: Foundations (Units 1-10) - Exercise Content

This module establishes the basic building blocks of English A1, focusing on personal information, identity, and immediate surroundings.

---

## Unit 1: Greetings & Goodbyes
**Focus**: Hello, Hi, Bye, See you. Verb 'To Be' (Positive).

### Exercises 1-10: Recognition
1. Match "Hello" with an image of people waving.
2. Match "Good morning" with a sun rising.
3. Match "Good night" with a moon.
4. Match "Goodbye" with someone leaving.
5. Audio: Hear "Hi" -> Select the informal greeting text.
6. Audio: Hear "Good afternoon" -> Select the correct time (2:00 PM).
7. Match "Nice to meet you" with two people shaking hands.
8. Identify the formal greeting: "Hello" vs "Hey".
9. Categorize: Greetings (Hi, Hello) vs Goodbyes (Bye, See you).
10. Match L1 greeting to English equivalent.

### Exercises 11-20: Concept Acquisition
11. Identify Verb 'To Be': "I ___ a student" (am/is/are).
12. Identify Verb 'To Be': "You ___ my friend" (am/is/are).
13. Logical sequence: "Hello" -> "How are you?" -> "____".
14. Spot the mistake: "I is happy" (is -> am).
15. Pattern matching: I am, You are, He/She/It is.
16. Formal vs Informal: Sort "Hi" and "Good morning".
17. Dialogue logic: "See you later" -> "___" (Bye/Hello).
18. Image-context: If it's 8:00 AM, say "____".
19. Personal pronoun matching: He (boy), She (girl), It (object).
20. Concept check: "Am" is only used with "I" (True/False).

### Exercises 21-35: Controlled Practice
21. Gap-fill: "I ___ John." (am)
22. Gap-fill: "She ___ a teacher." (is)
23. Sentence Scramble: [am / I / happy]
24. Sentence Scramble: [is / He / tall]
25. Gap-fill: "We ___ from Spain." (are)
26. Transformation: "I am" -> "I'm" (Contractions).
27. Transformation: "She is" -> "She's".
28. Gap-fill: "You ___ welcome." (are)
29. Sentence Scramble: [later / See / you]
30. Gap-fill: "Good ___ (evening/morning)" at 7:00 PM.
31. Negative check: "I am not" vs "I isn't".
32. Gap-fill: "It ___ a cat." (is)
33. Sentence Scramble: [a / It / book / is]
34. Transformation: "They are" -> "They're".
35. Gap-fill: "Nice to ___ you." (meet)

### Exercises 36-45: Functional Communication
36. Dialogue: A: "How are you?" B: "___" (I'm fine, thanks).
37. Dialogue: A: "I'm Sarah." B: "I'm Tom. ___" (Nice to meet you).
38. Situation: You arrive at a party. You say "___".
39. Situation: You leave work. You say "___".
40. Dialogue: A: "See you tomorrow." B: "___" (Yes, see you).
41. Response match: "Good morning" -> "Good morning".
42. Response match: "Goodbye" -> "Bye-bye".
43. Dialogue: A: "What's your name?" B: "___" (I'm Carlos).
44. Situation: You meet a teacher. Say "Good ___" (morning/hey).
45. Dialogue completion: [Hello / Hi / name / meet].

### Exercises 46-55: Active Production
46. Transcription: Hear "I am a teacher" -> Type it.
47. Transcription: Hear "See you later" -> Type it.
48. Translation: "Hola" -> "Hello".
49. Translation: "Yo soy" -> "I am".
50. Transcription: Hear "She is my friend" -> Type it.
51. Translation: "Adiós" -> "Goodbye".
52. Translation: "Mucho gusto" -> "Nice to meet you".
53. Transcription: Hear "How are you?" -> Type it.
54. Translation: "Él es" -> "He is".
55. Transcription: Hear "Good afternoon" -> Type it.

### Exercises 56-60: Synthesis & Mastery
56. Error fix: "You am my friend" -> "You are my friend".
57. Error fix: "He are happy" -> "He is happy".
58. Logic challenge: Put greetings in order from morning to night.
59. Context challenge: Choose the best greeting for a job interview.
60. Final Mastery: Complete a 4-line intro dialogue without hints.
