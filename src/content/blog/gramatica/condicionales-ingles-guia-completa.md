Los **condicionales en inglés** son estructuras que utilizamos para hablar de situaciones reales o imaginarias y sus consecuencias. En el nivel académico (B1 a C1), dominar estas formas es crucial para expresar hipótesis, arrepentimientos o verdades universales.

Como profesor de Cambridge, te presento esta guía estructurada para que entiendas no solo la gramática, sino la lógica detrás de cada tipo de condicional.

---

## 1. Zero Conditional: Verdades Universales
Se utiliza para hechos generales, leyes científicas o verdades que siempre ocurren.
*   **Estructura**: `If + Present Simple, ... Present Simple`
*   *Example: If you heat water to 100 degrees, it boils.*

---

## 2. First Conditional: Posibilidades Reales
Se usa para situaciones futuras que son muy probables o posibles.
*   **Estructura**: `If + Present Simple, ... Will + Infinitivo`
*   *Example: If it rains tomorrow, I will stay at home.*

---

## 3. Second Conditional: Sueños e Hipótesis
Se utiliza para hablar de situaciones imaginarias o poco probables en el presente o futuro. También se usa para dar consejos.
*   **Estructura**: `If + Past Simple, ... Would + Infinitivo`
*   *Example: If I won the lottery, I would buy a house in London.*
*   *Advice: If I were you, I would study harder.* (Recuerda que con 'I' usamos 'were' en condicionales).

---

## 4. Third Conditional: Arrepentimientos del Pasado
Se usa para hablar de situaciones que no ocurrieron en el pasado y sus consecuencias imaginarias. Es el condicional de los arrepentimientos.
*   **Estructura**: `If + Past Perfect, ... Would Have + Participio`
*   *Example: If I had studied more, I would have passed the exam.*

## 📥 Lista de 50+ Ejemplos para Práctica (Versión Imprimible)

### Zero Conditional
1.  If you touch fire, you get burned.
2.  If babies are hungry, they cry.
3.  Plants die if they don't get enough water.
4.  If you mix red and blue, you get purple.
5.  If it rains, the grass gets wet.

### First Conditional
6.  If I see him, I'll tell him the news.
7.  If you don't hurry, you will miss the bus.
8.  I will help you if you ask me.
9.  If we save enough money, we'll go on holiday.
10. If she arrives late, we will start without her.
11. If you eat too much, you'll feel sick.
12. They will stay at home if it snows.

### Second Conditional
13. If I were taller, I would play basketball.
14. If I had more time, I would travel more.
15. I would buy that car if I had the money.
16. If he knew the answer, he would tell us.
17. If it were sunny, we would go to the beach.
18. I would be happy if she came to the party.
19. If they lived closer, we would see them more often.
20. What would you do if you saw a ghost?

### Third Conditional
21. If I had known you were coming, I would have baked a cake.
22. She would have passed if she had studied.
23. If we hadn't missed the train, we wouldn't have been late.
24. If you had told me about the problem, I would have helped you.
25. They would have won if they had played better.
26. If he had driven more carefully, he wouldn't have had an accident.
27. I wouldn't have gone if I had known it would be so boring.

### Mixed & Advanced (Unless, As long as, Provided that)

29. You can stay **as long as** you are quiet.
30. I will lend you the book **provided that** you return it next week.

## Preguntas Frecuentes (FAQs)

### ¿Se puede cambiar el orden de la oración?
Sí. Puedes decir *If it rains, I'll stay* o *I'll stay if it rains*. La única diferencia es que si empiezas por "If", debes poner una coma antes de la segunda parte.

### ¿Se puede usar "Might" o "Could" en lugar de "Would" o "Will"?
¡Sí! Se usan para indicar que el resultado es menos seguro.
*Example: If I had money, I **might** buy a boat.*

### ¿Por qué se dice "If I were" y no "If I was"?
En el lenguaje formal y en exámenes Cambridge, se utiliza *were* para todas las personas en el Second Conditional (*If he were, If she were*). Indica que es una situación puramente hipotética.
