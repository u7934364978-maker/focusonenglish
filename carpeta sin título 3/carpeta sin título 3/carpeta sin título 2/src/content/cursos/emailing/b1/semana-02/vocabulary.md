# Vocabulary (B1) · Semana 02

> En esta semana priorizamos vocabulario útil para: **updates + follow-up + next steps**.

## 1) Palabras clave (EN → ES)
| EN | ES |
|---|---|
| update | actualización |
| status | estado |
| follow-up | seguimiento |
| next steps | siguientes pasos |
| timeline | cronograma |
| confirmation | confirmación |
| availability | disponibilidad |
| delay | retraso |

---

## 2) Colocaciones (muy típicas en emails)
| Colocación (EN) | Uso (ES) |
|---|---|
| any updates on… | ¿alguna novedad sobre…? |
| follow up on… | hacer seguimiento de… |
| confirm the timeline | confirmar el cronograma |
| move forward | avanzar / seguir adelante |
| as a next step | como siguiente paso |
| if you need more time | si necesitas más tiempo |

> “Any updates on…” + “as a next step” = combinación natural en emails.

---

## 3) Mini-frases listas (copy/paste)
- Just checking if you had a chance to review this.
- Do you have any updates on the approval?
- Would it be possible to confirm by Thursday?
- If you need more time, that’s fine.
- As a next step, I’ll share the final version.
- Once confirmed, I’ll proceed.
