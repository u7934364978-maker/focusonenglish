# Vocabulary (B1) · Semana 03

> En esta semana priorizamos vocabulario útil para: **scheduling + rescheduling + confirmations**.

## 1) Palabras clave (EN → ES)
| EN | ES |
|---|---|
| availability | disponibilidad |
| schedule | programar |
| reschedule | reprogramar |
| slot | hueco |
| calendar invite | invitación de calendario |
| meeting link | enlace de reunión |
| time zone | huso horario |
| conflict | conflicto (de agenda) |

---

## 2) Colocaciones (muy típicas en emails)
| Colocación (EN) | Uso (ES) |
|---|---|
| Does [time] work for you? | ¿Te va bien [hora]? |
| I’m available on… | Estoy disponible el… |
| send a calendar invite | enviar invitación |
| meeting conflict | conflicto de agenda |
| propose a time | proponer una hora |
| confirm the time | confirmar la hora |

> “Does … work for you?” = la frase más útil de la semana.

---

## 3) Mini-frases listas (copy/paste)
- Would Tuesday at 11:00 work for you?
- I’m available Wed 15:00 or Thu 10:30.
- Would it be possible to reschedule our call?
- Sorry for the change—something came up.
- Great — I’ll send a calendar invite.
- Please let me know which option works best.
