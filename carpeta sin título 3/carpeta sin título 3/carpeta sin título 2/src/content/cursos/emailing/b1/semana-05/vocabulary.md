# Vocabulary (B1) · Semana 05

> En esta semana priorizamos vocabulario útil para: **recap + decisions + action items + ownership**.

## 1) Palabras clave (EN → ES)
| EN | ES |
|---|---|
| recap | resumen |
| summary | resumen |
| action item | tarea / acción |
| owner | responsable |
| deadline | fecha límite |
| follow-up | seguimiento |
| decision | decisión |
| next steps | siguientes pasos |

---

## 2) Colocaciones (muy típicas en emails)
| Colocación (EN) | Uso (ES) |
|---|---|
| quick recap | resumen rápido |
| key points | puntos clave |
| action items | tareas/acciones |
| assign an owner | asignar responsable |
| agree on the timeline | acordar el cronograma |
| by Friday / by EOD Friday | para el viernes / fin de día viernes |

> “Quick recap + action items” = email súper profesional y útil.

---

## 3) Mini-frases listas (copy/paste)
- Here’s a quick recap of today’s call.
- Key points:
- Action items:
- Maria will share the numbers by Thursday.
- I’ll update the draft and send v3.
- If anything is missing, please let me know.
